# MyLevel4 Project

A full-stack web application built with Next.js and TypeScript, featuring authentication, post creation, and structured backend logic.

This project demonstrates real-world application architecture, including API handling, state management, and secure user interactions.

## Live Demo
https://mylevel4project.vercel.app

## Tech Stack
Next.js · TypeScript · MongoDB · Tailwind CSS · API Routes

## Features
- User authentication and session handling
- Create, update, and manage posts
- Improved error handling and validation
- Secure API routes with middleware/proxy logic
- Clean and scalable project structure

## Project Structure
- `app/` — Frontend pages and UI components
- `models/` — Database schemas and models
- `lib/` — Utility functions and configurations
- `proxy.ts` — Authentication and request handling

## Getting Started

```bash
npm install
npm run dev
